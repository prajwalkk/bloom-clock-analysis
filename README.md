# bloom-clock-analysis

Analysis of the simulations produced in the Bloom-clock-akka repo.

### Steps

1. clone this repo
2. clone the [bloom-clock-akka](https://github.com/prajwalkk/bloom-clock-akka) repo
3. Follow the instructions on the bloom-clock-akka repo to generate the simulations.
4. Use the notebooks in the notebooks folder to see some sample analyses.

### Issues

1. The file paths are absolute. Please change before trying to run the notebook.
